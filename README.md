# VibeTwist PRO - Live Canvas Animation

A high-performance React/Next.js application featuring interactive canvas animations powered by Framer Motion. Perfect for viral sharing, demos, and creative experiences.

## 🚀 Features

- ⚡ Real-time canvas animations
- 🎨 Beautiful gradient effects
- 🎬 Framer Motion integration
- 📱 Responsive design
- ⚙️ Server-side rendering with Next.js
- 🔧 Easy deployment to Vercel

## 📋 Prerequisites

- Node.js 14+ installed
- npm or yarn package manager
- GitHub account
- Vercel account (free)

## 🛠️ Installation

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/vibetwist-pro.git
   cd vibetwist-pro
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```
   
   Open [http://localhost:3000](http://localhost:3000) in your browser.

4. **Build for production**
   ```bash
   npm run build
   npm start
   ```

## 🌐 Deployment to Vercel

### Step 1: Push to GitHub
```bash
git remote add origin https://github.com/yourusername/vibetwist-pro.git
git branch -M main
git push -u origin main
```

### Step 2: Deploy on Vercel
1. Visit [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click "New Project"
4. Select "vibetwist-pro" repository
5. Click "Deploy"

Your app will be live in seconds! ✨

### Custom Domain (Optional)
1. In Vercel project settings
2. Add your domain under "Domains"
3. Follow DNS configuration
4. App is now on your custom domain
